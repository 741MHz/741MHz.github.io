Unique Lines Without Sorting

Today I ran into a small problem where I had a set of unsorted lines of text and wanted to remove duplicate lines without sorting the data. Both uniq and sort -u commands require data to be sorted, so they won’t work. There is a way of doing this with AWK:

awk '!($0 in lines) { lines[$0]; print }'

The above script will collect unique lines in order of their appearance and store them in array called lines that gets printed at the end. This probably won’t work for huge data sets but works well on up to a few thousand lines of text.
